📱 Android Modern Login UI
A clean, modern, and responsive Login Interface for Android applications. Designed with focus on user experience and visual appeal.

✨ Features
🎨 Modern Design: Beautifully crafted layout.

🌐 Social Authentication: Pre-designed buttons for Google, Facebook, and Twitter.

📱 Responsive: Works perfectly on different screen sizes.

🛠️ Custom Drawables: Includes high-quality icons.

🛠️ Technologies Used
XML - For the UI Layout.

Java - For backend logic.

Android Studio - Development Environment.

🚀 How to use
Clone this repository.

Open it in Android Studio.

Run it on an emulator or a physical device.
